# Logo placeholder

Replace this file with the official Uniputumayo logo:

1. Save the official logo as `logo_uniputumayo.png` inside this `assets/` folder.
2. Recommended size: at least 200 × 200 px, PNG with transparent background.
3. The app will automatically display it in the header if the file exists and is non-empty.

Until replaced, the app shows the 🎓 emoji as a stand-in.
